Nginx搭建教程

2020-08-07

Nginx是一个高性能的HTTP和反向代理的web服务站，同时也提供了IMAP/POP3/SMTP服务。

Mac上搭建Nginx

  • 安装Homebrew
  • 安装nginx
  • 验证结果
  1. 安装Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
  1. 安装nginx
brew install nginx
  1. 验证结果
    安装完成后，可以通过以下命令启动nginx
brew services start nginx

启动成功后，可以在浏览器访问地址: http://localhost:8080

其他说明

Nginx默认安装到 /usr/local目录下

Nginx重启命令

brew services restart nginx

Nginx停止命令

brew services stop nginx
